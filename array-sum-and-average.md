## Sum of arrays index value

```
num = 6
array = [1,2,3,4,5,6]
sum = 0

FOR i FROM 1 TO num-1:
    sum = sum + array[i]

PRINT sum
```

## Average of arrays index value

```
num = 6
array = [1,2,3,4,5,6]
sum = 0

FOR i FROM 1 TO num-1:
    sum = sum + array[i]

average = sum / num

PRINT average
```
